**Retail Sales Analysis Dashboard**
___
**Project Overview**

This project focuses on performing Exploratory Data Analysis (EDA) on a retail sales dataset using Microsoft Power BI. The dashboard provides insights into sales trends, customer purchasing behavior, product performance, and statistical analysis through interactive visualizations.

Developed as Task 1 for the Data Analytics Internship at Oasis Infobyte.

##
**Objectives**

Load and clean retail sales data
Perform descriptive statistical analysis
Analyze sales trends over time
Understand customer demographics and buying behavior
Visualize insights using interactive charts
Provide business recommendations based on EDA

##
**Features**

Data Loading & Cleaning
Removed null values and duplicates
Converted date columns into proper format
Created custom columns for analysis
Descriptive Statistics

##
**Calculated:**

Mean

Median

Mode

Standard Deviation

Total Sales

Time Series Analysis

Monthly sales trend analysis

Peak and low sales identification

Time-based sales visualization using line charts

Customer & Product Analysis

Gender-wise purchasing behavior

Age group analysis

Product category performance

Interactive Dashboard

KPI Cards

Line Charts

Bar Charts

Heatmap using Matrix Visual

Slicers and Filters

##
**Tools & Technologies**

Power BI

DAX

##
**Key Insights**

Electronics category generated the highest revenue

May month showed peak sales trends

Customers aged 46+ contributed most to sales

Sales patterns helped identify business opportunities

##
**Recommendations**

Increase stock for high-performing products

Provide discounts for low-performing categories

Target marketing campaigns based on customer demographics

Focus promotions during peak sales periods

##
**Dashboard Components**

Sales KPI Cards

Sales Trend Analysis

Customer Demographics

Product Category Analysis

Business Insights Section

Interactive Filters

##
**Conclusion**

The Retail Sales Analysis Dashboard helps businesses make data-driven decisions by identifying sales patterns, customer behavior, and product performance through effective visual analytics.
